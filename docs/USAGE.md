# Usage Guide

## Command Line
```bash
./ucmism2m \
  -input /path/to/input.uml \
  -output /path/to/output.uml \
  -config /path/to/config.json
```

## Configuration File

See `examples/config.json` for configuration options.

## Eclipse IDE

1. Open QVTo transformation file: `ucmism2m.transformation/transforms/m2m.qvto`
2. Right-click → Run As → QVTo Transformation
3. Configure input/output models and configuration in Run Configuration
