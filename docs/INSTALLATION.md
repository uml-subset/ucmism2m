# Installation Guide

## Prerequisites

- Java 21 or higher
- Maven 3.8+
- Eclipse Modeling Tools 2025-12 (for development)

## Building from Source

1. Clone the repository
2. Run `mvn clean verify`
3. Find the executable in `ucmism2m.product/target/products/`

## Eclipse IDE Setup

1. Import all projects as existing Maven projects
2. Set target platform to `ucmism2m.target`
3. Run QVTo transformations from Eclipse Run Configurations
